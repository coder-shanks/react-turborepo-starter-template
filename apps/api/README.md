# API (`apps/api`)

NestJS API app in this monorepo.

## Development

```bash
pnpm --filter api dev
```

## Build

```bash
pnpm --filter api build
```

## Start

```bash
pnpm --filter api start
pnpm --filter api start:prod
```

## Tests

```bash
pnpm --filter api test
pnpm --filter api test:e2e
pnpm --filter api test:cov
```
